# 24S-esc
Hi, I'm Anton and this is my project of creation a really powerfull ESC, made for drones able to lift a person 
In this repository or project you'll find scematics and tracing of my project, if you're interested in creating something similar whether it's a project or 
you're building a whole drone that needs to work with 100V battery, I'd be happy if my design will be helpfull, but I need to warn you to be carefull in using my design,
it's still a prototype and I will be definetly improving it and might forget to include some of improvments here, also 100V is super dangerous and can lead to really bad
consequences, so test and use it carefully, never forget about all the safety prosedures, test it with small voltage and current first, if changing parts make sure that they
can handle voltage spikes and at least don't touch it while connected to a 24S battery pack ;) 
Overall here I'll be documenting the building procces and testing of my ESC, and I'll share some usefull info on how you can create something similar, if you create something 
like that, with the help of some things I provided, pls share it with me, I'd really apreciate it, and again be carefull when testing it
